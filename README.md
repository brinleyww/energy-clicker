# Energy Clicker
A web-based incremental clicker game focused on **logic optimization** and **AI-assisted development.**

## Project Overview
Energy Clicker was developed over the course of the 2025-2026 school year as an exploration into **JavaScript**. The project focuses on efficient state management and iterative UI design.

## Key Features
* **Optimized Codebase:** Focused on keeping the final build under 150kb to ensure high performance and low latency across all browsers.
* **Iterative Design:** Refined through 25+ version updates, prioritizing bug mitigation and feature prototyping.
* **AI-Assisted Workflow:** Utilized AI tools to accelerate the prototyping of UI elements and to optimize core game-state calculations.
